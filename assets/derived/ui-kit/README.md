Source boards:

- `assets/new/buttons_new.png`
- `assets/new/samples.png`

Rules:

- Keep labels, counts, dates, and long copy in code.
- Use PNGs only for iconography, stickers, and reusable shell backgrounds.
- Prefer `shell_*` assets for layout chrome and `icon_*` assets for foreground actions.
