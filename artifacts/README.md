## Lightweight Windows desktop pet

- `PetFit-Buding-lite-desktop.zip`

Unzip it and run `start-petfit-lite.cmd`. This package uses Windows PowerShell and WinForms instead of Electron, so it stays small and behaves more like a lightweight floating desktop pet. Buding uses multiple PNG frames for idle, happy, snack, water, and nap states.
